# 계산기

## 지원 기능

| 기능                | 함수명         | 사용법(입력값)     | 설명                                                | 비고                    |
| ------------------- | -------------- | ------------------ | --------------------------------------------------- | ----------------------- |
| 덧셈(자연수)        | add            | 2 + 1              | a, b 두 수를 받아 합을 반환함                       | 자연수만 가능           |
| 뺄셈(자연수)        | subtract       | 6 - 4              | a, b 두 수를 받아 차를 반환함                       | 자연수만 가능           |
| 곱셈(자연수)        | multiply       | 7 \* 8             | a,b 두 수를 받아 곱한 값을 반환함                   | 자연수만 가능           |
| 나눗셈(자연수)      | divide         | 8 / 3              | a, b 두 수를 받아 a를 b로 나눈 값을 반환            | b > 0                   |
| 거듭제곱(자연수)    | power          | 4 \*\* 2           | a, b 두 수를 받아 a의 b제곱 값을 반환함             | 자연수만 가능, 0^0 불가 |
| 정수 나눗셈(자연수) | int_divide     | 4 // 3             | a, b 두 수를 받아 a를 b로 정수 나눗셈한 몫을 반환함 | b > 0                   |
| log 연산(자연수)    | log            | log_3(5)           | b를 밑으로 하는 a의 로그 값을 반환함                | a > 0, b > 0, b ≠ 1     |
| 미분(함수,변수)     | differentiator | x\*\*2+3 ↩ 3       | 함수와 변수를 받아 미분값을 반환함                  | 숫자만 가능             |
| 적분(함수,변수)     | integral       |                    | 함수를 받아 적분값을 반환함                         | 구현중...               |
| max 연산(자연수)    | max            | max(10, 20)        | a, b 중 큰 값을 반환함                              | 자연수만 가능           |
| min 연산(자연수)    | min            | min(10, 20)        | a, b 중 작은 값을 반환함                            | 자연수만 가능           |
| 증가함수            | increase       | 10++               | 넘겨 받은 수를 1 증가하여 반환함                    |                         |
| 감소함수            | decrease       | 11--               | 넘겨 받은 수를 1 감소하여 반환함                    |                         |
| 팩토리얼(자연수)    | factorial      | 10!, factorial(10) | 1부터 n까지의 곱을 반환함                           | 자연수만 가능           |
|조합(자연수)|combination|n개의 항목 중 r개를 선택하는 조합을 반환함|n>=0, r>=0, n>=r|
|순열(자연수)|permutation|n개의 항목 중 r개를 선택하는 순열을 반환함|n>=0, r>=0, n>=r|


### main.py

> 사용자가 원하는 연산 방식을 입력하고 연산하는 기능
> 사용법
> $ python main.py
> 시연 예

```shell
==================================================
※ 하고픈 연산을 선택해주세요. 제발요.
        1. 산술 연산
        2. 방정식 (미분/기울기)
        3. 수학 함수 연산
        9. 종료
==================================================
        ※ 메뉴 선택: 3

[ 수학 함수 연산 메뉴에 진입했습니다 ]
설명: 팩토리얼, 최댓값, 로그 등 수학 함수 형태의 연산입니다.

* 사용법 예시: max(10, 20)  또는  factorial(5)  또는  log(100, 10)
* 식을 입력해주세요 (이전 메뉴로 가려면 'q' 입력): 5!

        ===> 5! = 120


* 사용법 예시: max(10, 20)  또는  factorial(5)  또는  log(100, 10)
* 식을 입력해주세요 (이전 메뉴로 가려면 'q' 입력): factorial(2)

        ===> factorial(2) = 2
```
