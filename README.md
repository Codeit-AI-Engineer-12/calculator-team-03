# 계산기

## 지원 기능

|기능|함수명|설명|비고|
|-|-|-|-|
|덧셈(자연수)|add|a, b 두 수를 받아 합을 반환함|자연수만 가능|
|뺄셈(자연수)|subtract|a, b 두 수를 받아 차를 반환함|자연수만 가능|
|곱셈(자연수)|multiply|a,b 두 수를 받아 곱한 값을 반환함|자연수만 가능|
|나눗셈(자연수)|divide|a, b 두 수를 받아 a를 b로 나눈 값을 반환|b > 0|
|거듭제곱(자연수)|power|a, b 두 수를 받아 a의 b제곱 값을 반환함|자연수만 가능, 0^0 불가|
|정수 나눗셈(자연수)|int\_divide|a, b 두 수를 받아 a를 b로 정수 나눗셈한 몫을 반환함|b > 0|
|log 연산(자연수)|log|b를 밑으로 하는 a의 로그 값을 반환함|a > 0, b > 0, b ≠ 1|
|미분(함수,변수)|differentiator|함수와 변수를 받아 미분값을 반환함|숫자만 가능|
|max 연산(자연수)|max|a, b 중 큰 값을 반환함|자연수만 가능|
|증가함수|increase|넘겨 받은 수를 1 증가하여 반환함||
|감소함수|decrease|넘겨 받은 수를 1 감소하여 반환함||
|팩토리얼(자연수)|factorial|1부터 n까지의 곱을 반환함|자연수만 가능|
|조합(자연수)|combination|n개의 항목 중 r개를 선택하는 조합을 반환함|n>=0, r>=0, n>=r|
